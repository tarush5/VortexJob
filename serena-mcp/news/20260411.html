Serena v1.1: Hooks and New Release Mechanism

April 11, 2026

Installation. Serena has matured, and we are switching to a PyPI-based release and installation mechanism. The recommended uv tool-based installation decreases startup times and facilitates CLI usage. See our updated installation instructions for more details, and update your client configurations after having switched to the new installation method.

Improving Tool Usage via Hooks. To address issues with agents not using Serena's tools (agent drift), we now provide the `serena-hooks` commands and a recommended setup for Claude Code and VSCode. Please read the updated documentation for more details. We strongly recommend using the hooks, since recent updates in Claude Code and VSCode resulted in reduced performance, which our hook commands compensate for.

Community. We opened a Discord server for our community.
