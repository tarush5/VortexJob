Serena v1.3.0

May 12, 2026

Diagnostics & Inspections. New retrieval tools and inspection/diagnostics tools for both backends.

  • LSP: find_declaration, find_implementations, get_diagnostics_for_file, and get_diagnostics_for_symbol
  • JetBrains: jet_brains_list_inspections and jet_brains_run_inspections

Expanded Language Support. Five new languages to Serena's LSP backend: Ada/SPARK, Angular, HTML, SCSS/Sass/CSS, and 1C/OneScript.

Changes in Mode Configuration. Projects can no longer override base_modes in project.yml but can now use the new added_modes key for more flexible configuration. See the updated configuration documentation for details.

Improved Reminder Hook Logic. The hooks have been improved and now also react to Codex' bash tool.

See our change log for the full list of improvements.

Update Now. If you are using the uv tool installation of Serena, upgrade as follows:
uv tool upgrade serena-agent --prerelease=allow
